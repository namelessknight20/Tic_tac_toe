# 🎮 Tic-Tac-Toe (Player vs AI)

A simple and interactive Tic-Tac-Toe game where a human player competes against an AI opponent. Built using HTML and JavaScript, this project demonstrates basic game logic and AI decision-making.

---

## 🚀 Features

- 🧑‍💻 Player vs AI gameplay
- 🤖 Smart AI ( Minimax-based )
- 🎨 Simple and clean UI
- ⚡ Fast and lightweight (runs directly in browser)
- 🔁 Restart game functionality

---

## 🛠️ Tech Stack

- **HTML** – Structure of the game (`index.html`)
- **JavaScript** – Game logic and AI (`sketch.js`)


---

## 🎯 How to Play

1. Open `index.html` in your browser
2. You play as **X** (or as defined in your code)
3. Click on any empty cell to make your move
4. The AI will automatically respond
5. First to get 3 in a row wins!

---

## 🧠 AI Logic

The AI in this game:
- Chooses moves based on predefined logic
  Uses the **Minimax Algorithm** to always play the optimal move
